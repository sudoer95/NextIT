A NextJS E-commerce tech shop.

## Getting Started

First, clone the repo.
Then run following commands, depending on your needs:

```bash
npm install
# or
yarn install
# or
pnpm i
```

```bash
npm run dev
# or
yarn dev
# or
pnpm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
